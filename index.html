WatBus Android Application

WatBus is an Android application developed using Android Studio with a friend. The user selects a campus building and the closest bus stop to the campus building is displayed. The application also provides the bus stop number which they can text to 575-55 to find the next scheduled bus at that stop. It provides the user the convenience of knowing the updated bus times before going to the stop. This was made in a 2 person team, which allowed me to become more comfortable working with a Version Control System (Git).

  • Source code
  • Date: December 2015
